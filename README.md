# 🏠 Airbnb Global Performance Dashboard | Power BI Project


# 📊 Project Overview

The **Airbnb Global Performance Dashboard** is an interactive Power BI project designed to analyze global Airbnb listings, customer reviews, pricing trends, and seasonal demand patterns across multiple cities.

This project demonstrates how raw data can be transformed into meaningful insights using data visualization and business intelligence tools.

---

# 🎯 Project Objectives

The main objectives of this project are:

- Analyze Airbnb listing growth trends over time  
- Compare market share across major cities  
- Evaluate customer satisfaction through ratings  
- Identify seasonal travel and booking patterns  
- Analyze host trust and identity verification trends  
- Create an interactive dashboard for business insights

# 📊 Dashboard Features

## 📈 Listings Growth Analysis

- Tracks listing growth from **2008 to 2020**
- Identifies peak growth periods
- Highlights slowdown during **COVID-19**

---

## 🌍 Market Share by City

- Compares listing distribution across cities
- Identifies top-performing markets
- Major cities analyzed include:

  - Paris  
  - New York  
  - Sydney  
  - Rome  
  - Mexico City  

---

## 💰 Price Analysis by Property Type

Analyzed pricing trends across:

- Entire Place  
- Hotel Room  
- Private Room  
- Shared Room  

This helps understand pricing differences between property types.

---

## ⭐ Customer Ratings Analysis

Evaluated customer ratings across:

- Accuracy  
- Cleanliness  
- Communication  
- Location  
- Value  

Top-rated cities identified based on performance metrics.

---

## 📅 Seasonality Analysis

Identified peak demand months:

- European cities peak during **April to August**
- New York shows increased activity in **November–December**

This helps understand seasonal demand patterns.

---

## 🔐 Trust & Verification Analysis

Analyzed host trust indicators:

- Identity Verification Status  
- Profile Completion  
- Trust Signals  

Majority of hosts are verified, indicating strong platform reliability.

---

# 📈 Key Insights

Some important findings from the dashboard include:

- Paris, New York, and Sydney contribute nearly **50% of total listings**  
- Mexico City and Rio de Janeiro show **highest customer ratings**  
- European cities experience peak demand during **summer months**  
- Most customers leave **1–3 reviews**, indicating repeat usage patterns  
- Majority of hosts have verified identities
